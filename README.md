# 📊 Power BI Dashboard Project

## 📌 Project Overview

This project is an interactive **Power BI Dashboard** developed to provide business insights, KPI monitoring, trend analysis, and operational reporting.

The dashboard enables users to:

* Monitor key performance indicators (KPIs)
* Analyze trends and business performance
* Filter and drill down into detailed data
* Generate data-driven insights for decision-making



# 🧩 Features

* Interactive dashboards
* KPI Cards & Gauges
* Dynamic slicers and filters
* Trend analysis charts
* Pie/Donut category analysis
* Drill-down reporting
* Tabular data visualization
* Multi-page reporting structure



# 🛠️ Tech Stack

| Technology       | Purpose                 |
| ---------------- | ----------------------- |
| Power BI Desktop | Dashboard Development   |
| DAX              | Calculations & Measures |
| Data Modeling    | Relationship Management |
| Power Query      | Data Transformation     |
